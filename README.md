## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/ShaanifFaqui/atomnext.git
```

2. Install dependencies:
```bash
cd atomnext
pip install -r requirements.txt
```

3. Run the development server:
```bash
python app.py
```

4. Visit `http://localhost:5000` in your browser
